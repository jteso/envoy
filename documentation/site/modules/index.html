Overview

Envoy ships with out-of-the-box modules that can be attached to your API policy in order to augment your API. The main mission of these modules are to enhance peformance, enforce security, or increase the utility of your APIs, without requiring you to modify any upstream endpoint.

Following are some of the modules that can be found:

Security

  • Access
  • Basic Authentication
  • CORS

Traffic Management

  • Gzip
  • Spike Arrest

Extensions

Enpoints

  • HTTP Router
